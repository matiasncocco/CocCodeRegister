CocCodeRegister es un programa Basico de registro y consulta

ANTES DE EJECUTAR LA APLICACION CoccodeRegister:

- Instalar el interprete 'python-3.12.0-amd64':

   Marcar la opcion 'Add Python.exe to PATH'
   
   Marcar todas las opciones en CUSTOMIZE INSTALLATION y luego de presionar 
   NEXT, marcar todos los check en ADVANCE OPTIONS. 

   Presionar INSTALL

   

- En buscador de Windows escribir "Windows Powershell" ,abrirlo:
  
   Escribir ---->  cd .\Desktop\CoccodeRegister\  ------> ENTER

   Escribir ---->  pip install tk ------> ENTER
   
   Escribir ----> pip install openpyxl ------> ENTER


LISTO, ejecutar!
